📄️ 媒体查询

CSS 2.1 中定义了各种媒体类型，包括显示器、便携设备、电视机等。 CSS 3 增加了 Media Queries 模块，该模块允许添加指定的媒体查询表达式，用以指定媒体类型，然后根据媒体类型来选择样式。

📄️ 自定义字体

@font-face 属性不仅可以用于显示服务器端的字体，也可以用来显示客户端本地的字体。

📄️ 预处理

CSS 并不是一门真正意义上的编程语言，因为它缺少变量、运算、函数和控制语句等编程语言应该有的语法，这造成 CSS 代码不但难以复用，而且在 CSS 文件中充斥着冗余的定义，随着项目规模的扩大，维护复杂度骤增。
