nvm

在最近一次安装 node ，发现官方推荐使用 nvm

安装​

使用推荐大方法安装 nvm

# Download and install nvm:
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
# in lieu of restarting the shell
\. "$HOME/.nvm/nvm.sh"

安装完毕后需要将环境添加到启动环境 ，在 ~/.zshrc 文件中添加

# 加载 NVM
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"  # 加载 nvm
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"  # 加载 nvm bash 补全

列出当前的版本​

使用 ls 命令列出当前的使用版本的信息

列出可用版本​

使用 ls-remote 列出可用的版本信息

安装新的版本​

使用 install 安装上面使用 ls-remote 列出的可用版本

别名设置​

使用 alias 进行版本的别名设置。

命名为 default 的命名为默认的当前版本。
