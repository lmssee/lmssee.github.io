DOM дёӯзҡ„жіЁйҮҠйҖҡиҝҮ Comment зұ»еһӢиЎЁзӨәгҖӮ

  • nodeType зӯүдәҺ 8
  • nodeName еҖјдёә"#comment"
  • nodeValue еҖјдёәжіЁйҮҠзҡ„еҶ…е®№
  • parentNode еҖјдёә Document жҲ– Element еҜ№иұЎ
  • дёҚж”ҜжҢҒеӯҗиҠӮзӮ№

Comment зұ»еһӢдёҺ Text зұ»еһӢз»§жүҝеҗҢдёҖдёӘеҹәзұ»пјҲ CharacterData пјүпјҢеӣ жӯӨжӢҘжңүйҷӨ splitText()д№ӢеӨ– Text иҠӮзӮ№жүҖжңүзҡ„еӯ—з¬ҰдёІж“ҚдҪңж–№жі•гҖӮдёҺ Text зұ»еһӢзӣёдјјпјҢжіЁйҮҠзҡ„е®һйҷ…еҶ…е®№еҸҜд»ҘйҖҡиҝҮ nodeValue жҲ– data еұһжҖ§иҺ·еҫ—гҖӮ

CDATASectionвҖӢ

CDATASection зұ»еһӢиЎЁзӨә XML дёӯзү№жңүзҡ„ CDATA еҢәеқ—гҖӮ CDATASection зұ»еһӢз»§жүҝ Text зұ»еһӢпјҢеӣ  жӯӨжӢҘжңүеҢ…жӢ¬ splitText()еңЁеҶ…зҡ„жүҖжңүеӯ—з¬ҰдёІж“ҚдҪңж–№жі•гҖӮ CDATASection зұ»еһӢзҡ„иҠӮзӮ№е…·жңүд»ҘдёӢзү№еҫҒгҖӮ

  • nodeType зӯүдәҺ 4
  • nodeName еҖјдёә"#cdata-section
  • nodeValue еҖјдёә CDATA еҢәеқ—зҡ„еҶ…е®№
  • parentNode еҖјдёә Document жҲ– Element еҜ№иұЎ
  • дёҚж”ҜжҢҒеӯҗиҠӮзӮ№

CDATA еҢәеқ—еҸӘеңЁ XML ж–ҮжЎЈдёӯжңүж•ҲпјҢеӣ жӯӨжҹҗдәӣжөҸи§ҲеҷЁжҜ”иҫғйҷҲж—§зҡ„зүҲжң¬дјҡй”ҷиҜҜең°е°Ҷ CDATA еҢәеқ—и§Јжһҗ дёә Comment жҲ– Element гҖӮ
