еҲ©з”Ё localStorage еӯҳеӮЁ JSON еҜ№иұЎ

еҲ©з”Ё localStorage еҸҜд»Ҙз®ҖеҚ•ең°еӯҳеӮЁдёҖдәӣ JSON еҜ№иұЎпјҢеҸҜд»Ҙж–№дҫҝз®Җжҳ“ең°еә”з”Ёж•°жҚ®еӯҳеӮЁгҖӮжҲ‘д»¬жіЁж„ҸеҲ° localStorage еӯҳеӮЁзҡ„еҖјйғҪжҳҜеӯ—з¬ҰдёІзұ»еһӢпјҢеңЁеӨ„зҗҶеӨҚжқӮзҡ„ж•°жҚ®ж—¶пјҢйңҖиҰҒеҖҹеҠ© JSON зұ»пјҢе°Ҷ JSON еӯ—з¬ҰдёІиҪ¬жҚўжҲҗзңҹжӯЈеҸҜз”Ёзҡ„ JSON ж јејҸгҖӮ

дҪҝз”Ё JSON.stringify() ж–№жі•е°Ҷ JSON еҜ№иұЎиҪ¬жҚўдёәеӯ—з¬ҰдёІеҗҺпјҢеҶҚиҝӣиЎҢеӯҳеӮЁпјҢеҚіеҸҜдҝқеӯҳ JSON еҜ№иұЎ . дҪҝз”Ё JSON.parse() ж–№жі•е°ҶеҸ–еҮәзҡ„еӯ—з¬ҰдёІиҪ¬жҚўдёә JSON еҜ№иұЎпјҢеҚіеҸҜеҫ—еҲ°еӯҳеӮЁзҡ„ students зҡ„ JSON еҜ№иұЎгҖӮ

еҲ©з”Ё localStorage и®°еҪ•з”ЁжҲ·иЎЁеҚ•иҫ“е…ҘвҖӢ

еҸҜд»ҘдҪҝз”Ё localstorage и®°еҪ•з”ЁжҲ·иЎЁеҚ•иҫ“е…ҘпјҢеҚідҪҝе…ій—ӯжөҸи§ҲеҷЁпјҢдёӢж¬ЎйҮҚж–°жү“ејҖжөҸи§ҲеҷЁи®ҝй—®пјҢд№ҹиғҪиҜ»еҮәиЎЁеҚ•дёӯиҫ“е…Ҙзҡ„еҖјпјҢйҒҝе…Қз”ЁжҲ·йҮҚж–°иҫ“е…ҘпјҢиҝҷеҸҜд»Ҙз»ҷз”ЁжҲ·еёҰжқҘжӣҙеҠ е®Ңе–„зҡ„дҪ“йӘҢж–№ејҸгҖӮ

еҲ©з”Ё localStorage иҝӣиЎҢи·Ёж–ҮжЎЈж•°жҚ®дј йҖ’вҖӢ

localStorage иҝҳжҸҗдҫӣдәҶдёҖдёӘ storage дәӢд»¶пјҢеңЁеӯҳеӮЁзҡ„еҖјж”№еҸҳеҗҺи§ҰеҸ‘гҖӮзӣ‘еҗ¬ storage дәӢд»¶пјҢеҸҜд»ҘиҺ·еҸ–ж”№еҸҳзҡ„еӯ—ж®өгҖҒж”№еҸҳеӯ—ж®өзҡ„ж—§еҖје’Ңж–°еҖјгҖӮйҖҡиҝҮ storage дәӢд»¶пјҢеҸҜд»Ҙе®һзҺ°и·Ёж–ҮжЎЈж•°жҚ®дј йҖ’гҖӮ

еңЁ localStorage дёӯеӯҳеӮЁеӣҫзүҮвҖӢ

жң¬ең°еӯҳеӮЁеҸӘж”ҜжҢҒеӯ—з¬ҰдёІзҡ„еӯҳеҸ–пјҢйӮЈд№ҲжҲ‘д»¬иҰҒеҒҡзҡ„е°ұжҳҜе°ҶеӣҫзүҮиҪ¬жҚўжҲҗ Data URI гҖӮе…¶дёӯдёҖз§Қе®һзҺ°ж–№ејҸе°ұжҳҜз”Ё canvas е…ғзҙ жқҘеҠ иҪҪеӣҫзүҮпјҢе°Ҷ Data URI зҡ„еҪўејҸд»Һ canvas дёӯиҜ»еҸ–еҮәеҪ“еүҚеұ•зӨәзҡ„еҶ…е®№гҖӮ

еңЁ localStorage дёӯеӯҳеӮЁж–Үд»¶вҖӢ

дҝқеӯҳд»»ж„Ҹж јејҸзҡ„ж–Үд»¶еҹәжң¬жҖқи·ҜжҳҜйҰ–е…ҲдҪҝз”Ё XMLHttpRequest иҜ·жұӮж–Үд»¶пјҢз„¶еҗҺе°Ҷе“Қеә”еӨҙи®ҫзҪ®дёә "arrayBuffer" пјӣе…¶ж¬Ўе°Ҷиҝ”еӣһж•°жҚ®еӯҳгҖӮ

дҪҝз”Ё localForage иҝӣиЎҢзҰ»зәҝеӯҳеӮЁвҖӢ

localStorage иғҪеӨҹе®һзҺ°еҹәжң¬зҡ„ж•°жҚ®еӯҳеӮЁпјҢдҪҶжҳҜеҜ№еӨҚжқӮж•°жҚ®зҡ„еӨ„зҗҶиҝҮзЁӢжҜ”иҫғеӨҚжқӮгҖӮйҰ–е…ҲпјҢ localStorage жҳҜеҗҢжӯҘзҡ„пјҢдёҚи®әж•°жҚ®еӨҡеӨ§пјҢйғҪйңҖиҰҒзӯүеҫ…ж•°жҚ®д»ҺзЈҒзӣҳиҜ»еҸ–е’Ңи§ЈжһҗпјҢиҝҷдјҡеҮҸж…ўеә”з”ЁзЁӢеәҸзҡ„е“Қеә”йҖҹеәҰпјҢиҝҷеңЁз§»еҠЁи®ҫеӨҮдёҠжҳҜзү№еҲ«зіҹзі•зҡ„пјҢдё»зәҝзЁӢиў«жҢӮиө·пјҢзӣҙеҲ°ж•°жҚ®иў«еҸ–еҮәпјҢдјҡдҪҝдҪ зҡ„еә”з”ЁзЁӢеәҸзңӢиө·жқҘж…ўпјҢз”ҡиҮіжІЎжңүеҸҚеә”гҖӮжӯӨеӨ–пјҢ localStorage д»…ж”ҜжҢҒеӯ—з¬ҰдёІпјҢйңҖиҰҒдҪҝз”Ё JSON.parse дёҺ JSON.stringify иҝӣиЎҢеәҸеҲ—еҢ–е’ҢеҸҚеәҸеҲ—еҢ–гҖӮиҝҷжҳҜеӣ дёә localStorage дёӯд»…ж”ҜжҢҒ JavaScript еӯ—з¬ҰдёІеҖјпјҢдёҚж”ҜжҢҒж•°еҖјгҖҒеёғе°”еҖјгҖҒ Blob зұ»еһӢзҡ„ж•°жҚ®гҖӮ Mozilla ејҖеҸ‘дёҖдёӘеҗҚдёә localForage зҡ„еә“пјҢдҪҝеҫ—зҰ»зәҝж•°жҚ®еӯҳеӮЁеңЁд»»дҪ•жөҸи§ҲеҷЁеӯҳеӮЁйғҪйқһеёёж–№дҫҝе’Ңжҳ“з”ЁгҖӮ

localForage жҳҜдёҖдёӘдҪҝз”Ёйқһеёёз®ҖеҚ•зҡ„ JavaScript еә“пјҢжҸҗдҫӣдәҶ get гҖҒ set гҖҒ remove гҖҒ clear е’Ң length зӯү API пјҢиҝҳе…·жңүд»ҘдёӢзү№зӮ№ гҖӮ

  • ж”ҜжҢҒ BLOB е’Ңд»»ж„Ҹзұ»еһӢзҡ„ж•°жҚ®пјҢеҸҜд»ҘеӯҳеӮЁеӣҫзүҮгҖҒж–Үд»¶зӯү

йҳІжӯўж•°жҚ®жі„йңІвҖӢ

еҜ№дәҺи®ҫзҪ®еңЁ sessionStorage дёӯзҡ„еҜ№иұЎпјҢеҸӘиҰҒжөҸи§ҲеҷЁзӘ—еҸЈпјҲжҲ–ж ҮзӯҫпјүдёҚе…ій—ӯпјҢе®ғд»¬е°ұдјҡдёҖзӣҙеӯҳеңЁгҖӮеҪ“з”ЁжҲ·е…ій—ӯзӘ—еҸЈжҲ–жөҸи§ҲеҷЁж—¶пјҢ sessionStorage ж•°жҚ®е°Ҷиў«жё…йҷӨгҖӮжүҖд»ҘејҖеҸ‘дәәе‘ҳдёҚеә”иҜҘжҠҠзңҹжӯЈжңүд»·еҖјзҡ„дёңиҘҝж”ҫеңЁйҮҢйқўпјҢеӣ дёәдёҚиғҪд»»дҪ•ж—¶еҖҷжҹҘиҜўиҝҷдәӣж•°жҚ®йғҪеӯҳеңЁгҖӮ

sessionStorage йқһеёёйҖӮеҗҲз”ЁдәҺзҹӯж—¶еӯҳеңЁзҡ„жөҒзЁӢдёӯпјҢеҰӮеҜ№иҜқжЎҶе’Ңеҗ‘еҜјгҖӮеҰӮжһңж•°жҚ®йңҖиҰҒеӯҳеӮЁеңЁеӨҡдёӘйЎөйқўдёӯпјҢеҗҢж—¶еҸҲдёҚеёҢжңӣз”ЁжҲ·дёӢдёҖж¬Ўи®ҝй—®еә”з”ЁзЁӢеәҸж—¶йҮҚж–°йғЁзҪІпјҢеҲҷеҸҜе°Ҷиҝҷдәӣж•°жҚ®еӯҳеӮЁеңЁ sessionStorage дёӯгҖӮд»ҘеүҚпјҢиҝҷзұ»ж•°жҚ®еҸҜиғҪйңҖиҰҒйҖҡиҝҮиЎЁеҚ•е’Ң cookie жҸҗдәӨпјҢе№¶еңЁйЎөйқўеҠ иҪҪж—¶жқҘеӣһдј йҖ’пјҢиҖҢдҪҝз”Ё Storage еҸҜд»ҘйҒҝе…Қиҝҷз§ҚејҖй”ҖгҖӮ

sessionStorage иҝҳжңүеҸҰеӨ–дёҖз§Қзү№ж®Ҡз”Ёжі•пјҡж•°жҚ®дҪңз”ЁеҹҹгҖӮе®ғи§ЈеҶідәҶдёҖдёӘдёҖзӣҙеӯҳеңЁдәҺиҜёеӨҡ Web еә”з”ЁзЁӢеәҸзҡ„й—®йўҳпјҢд»Ҙиҙӯд№°жңәзҘЁзҡ„иҙӯзү©еә”з”ЁзЁӢеәҸдёәдҫӢпјҢеңЁиҝҷдёӘеә”з”ЁзЁӢеәҸдёӯпјҢиҜёеҰӮзҗҶжғізҡ„еҮәеҸ‘иҝ”еӣһж—Ҙжңҹиҝҷж ·зҡ„з”ЁжҲ·еҒҸеҘҪж•°жҚ®пјҢеҸҜиғҪдјҡеңЁжөҸи§ҲеҷЁе’ҢжңҚеҠЎеҷЁй—ҙдҪҝз”Ё cookie жқҘеӣһдј йҖҒпјҢдёәзҡ„жҳҜеңЁз”ЁжҲ·дҪҝз”Ёеә”з”Ёж—¶пјҢжңҚеҠЎеҷЁеә”з”ЁзЁӢеәҸиғҪи®°дҪҸз”ЁжҲ·е…ҲеүҚйҖүжӢ©зҡ„еҒҸеҘҪж•°жҚ®гҖӮ

дёҚиҝҮпјҢз”ЁжҲ·жү“ејҖеӨҡдёӘзӘ—еҸЈжҳҜеҫҲеёёи§Ғзҡ„пјҢе®ғд»¬еҸҜиғҪеңЁжҹҘзңӢж—…жёёдә§е“Ғж—¶еҗҢж—¶жү“ејҖеӨҡдёӘзӘ—еҸЈпјҢжҜ”иҫғдёҚеҗҢд»ЈзҗҶе•ҶеҗҢдёҖж—¶й—ҙиө·йЈһзҡ„иҲӘзҸӯгҖӮиҝҷдјҡеҜјиҮҙ cookie зі»з»ҹеҮәзҺ°й—®йўҳпјҢеӣ дёәеҰӮжһңдёҖдёӘз”ЁжҲ·еңЁжҜ”иҫғд»·ж је’ҢжҳҜеҗҰжңүзҘЁзӯүдҝЎжҒҜж—¶дјҡеңЁжөҸи§ҲеҷЁзӘ—еҸЈд№Ӣй—ҙжқҘеӣһеҲҮжҚўпјҢеҫҲеҸҜиғҪдјҡеңЁе…¶дёӯдёҖдёӘзӘ—еҸЈи®ҫзҪ® cookie еҖјпјҢиҖҢеңЁе…¶е®ғзӘ—еҸЈдёӯж„ҸеӨ–ең°е°ҶиҝҷдәӣеҖјеә”з”ЁеҲ° URL зӣёеҗҢзҡ„еҸҰдёҖдёӘзҪ‘йЎөзҡ„еҗҺз»ӯж“ҚдҪңдёӯгҖӮиҝҷдёҖзҺ°иұЎд№ҹиў«з§°дёәж•°жҚ®жі„йңІгҖӮе…¶дә§з”ҹзҡ„ж №жң¬еҺҹеӣ еңЁдәҺ cookie иғҪеӨҹиў«еҗҢжәҗзҪ‘йЎөе…ұдә«гҖӮ

иҖҢдҪҝз”Ё sessionStorage иғҪеӨҹи·ЁйЎөйқўпјҲдҪҝз”ЁиҜҘеә”з”Ёзҡ„йЎөйқўпјүжҡӮеӯҳеҰӮеҗҜзЁӢж—Ҙжңҹиҝҷж ·зҡ„дёҙж—¶ж•°жҚ®пјҢеҸҲдёҚдјҡе°Ҷе…¶жі„йңІеҲ°з”ЁжҲ·д»ҚеңЁжөҸи§Ҳзҡ„е…¶е®ғиҲӘзҸӯдҝЎжҒҜзҡ„зӘ—еҸЈдёӯгҖӮиҝҷж ·пјҢдёҚеҗҢзҡ„еҒҸеҘҪдҝЎжҒҜе°ұдјҡиў«йҡ”зҰ»еңЁйў„и®ўзӣёеә”иҲӘзҸӯзҡ„зӘ—еҸЈдёӯгҖӮ
