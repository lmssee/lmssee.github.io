实体字符

为了能在 HTML 文档中正确显示某些特殊字符，就需要使用 HTML 实体（ entity ）。 HTML 实体就是对当前文档的编码方式不能包含的字符，提供一种转义表示。

HTML 实体的定义 :

名称方式​

名称方式会以" & "开头，然后紧跟由英文字母组成的名称（下面格式中的 name ），最后以分号结尾。注意，这种方式对大小写敏感

十进制方式​

十进制方式会以" &# "开头，然后紧跟数字（下面格式中的 number ），最后以分号 结尾

十六进制方式​

十六进制方式会以" &#x "开头，然后紧跟十六进制数（下面格式中的 hex ），最后以分号结尾

字符实体​

可转义的字符或符号总共有五类 。

  • ASCII 字符， ASCII 字符共 128 个字符，其中有 33 个字符无转义，分别是 0 ~ 31 号以及 127 号字符
  • ISO 8859-1 字符和符号，以 ASCII 为标准，加入 88 个字母以及符号，支持部分欧洲使用语言
  • 数学符号，在数学表达式中出现的符号，包括 ∑、∞、≈ 等各种专用符号
  • 希腊字母，希腊语所使用的字母，广泛用于数学、物理等学科，包括 α、β、Ω 等
