列表

  • 有序列表：一种在每个列表项前具有数字或字母的缩进列表。有序列表开始于 <ol> ，结束于 </ol> 封闭标签。列表项封闭在 <li></li> 标签对内，并且在每个 <li> 开始标签处会自动出现换行符。整个列表是缩进的
  • 无序列表：一种在每个列表项前具有项目符号或其它符号的缩进列表。无序列表开始于 <ul>标签，并且结束于 </ul> 。与有序列表一样，它的列表项封闭在 <li></li> 标签对内。换行符和符号出现在每个 <li> 开始标签处，并且整个列表是缩进的
  • 定义列表：术语及其意义的列表。这类列表在每个列表项前没有特殊的数字 、 字母或符号，它开始于 <dl> ，并结束于 </dl> 。
  • <dt></dt> 标签对封闭每个列表项， <dd></dd> 标签对则封闭每个定义

有序列表​

  • ol 一个有序列表
  • li 列表项

无序列表​

  • ul 一个无序列表
  • li 列表项

定义列表​

  • dl 定义一个定义列表
  • dt 定义术语 , 作为定义列表的一部分
  • dd 定义术语对应的一部分 , 作为定义列表的一部分

定义样式 , 详见 css 。

菜单列表​

<menu> 标签可以包括 <command> 、 <menuitem> 标签，用于定义命令和项目。

属性值嗯
checkedchecked定义在页面中加载后选中菜单 / 命令项目。仅用于 type="radio" 或 type="checkbox"
defaultdefault把命令 / 菜单项设置为默认命令
disableddisabled定义命令 / 菜单项是否应该被禁用
iconURL定义菜单 / 命令的图标
openopen定义 details 是否可见
labeltext必须。定义命令 / 菜单的名称，以向用户显示
radiogroupgroup name定义命令组名称，命令组会在命令 / 菜单项本身被切换时切换。仅适用于 type="radio"
typecheckbox 、 command 、 radio定义该 command 的类型，默认为 command

<menu> 标签有两个专用的属性

  • label 定义菜单的可见标签

  • type 定义要显示哪种菜单类型

  • list 定义列表菜单。一个用户可执行或激活的命令菜单

  • context 定义上下文菜单。该菜单必须在用户能够与命令交互前激活

  • toolbar 定义工具栏菜单。活动式命令，允许用户立即与命令进行交互

快捷菜单​

menuitem 标签用来定义菜单项目，这些菜单仅用于弹出菜单命令，方便用户快捷调用。

属性值嗯
checkedchecked定义是否被选中，仅用于 radio 和 checkbox
disableddisabled定义 command 是否可用
iconURL定义用作 command 的图像的 URL
labeltext为 command 定义可见的 label
radiogroupgroupName定义 command 所属的组名，仅在类型为 radio 时可用
typecheckbox 、 command 、 radio定义该 command 的类型，默认为 command
<div contextmenu="my">
  上下文菜单
  <menuitem label=" 微信分享 "> </menuitem>
  <menuitem label=" 微博分享 "> </menuitem>
</div>
